# M3F-MECPE

This repository provides resources for our ACM Multimedia (MM) paper on multilingual multimodal emotion–cause analysis with a memory-inspired framework.

## Paper

**Emotion across modalities and cultures: Multilingual multimodal emotion-cause analysis with memory-inspired framework**  
Dan Wu, Xincheng Ju, Dong Zhang, Shoushan Li, Erik Cambria, Guodong Zhou  
**ACM Multimedia (MM) 2025**, Proceedings of the 33rd ACM International Conference on Multimedia, pp. 5775–5783

- DOI (Version of Record): https://doi.org/10.1145/3746027.3755655

## Repository Contents

- `paper.pdf` (optional): author-prepared version of the paper (recommended to upload if allowed).
- `code/` (planned): training/inference code and configs.
- `data/` (planned): dataset release or instructions for obtaining/building the dataset.
- `scripts/` (planned): preprocessing and evaluation utilities.

> Note: If the dataset cannot be redistributed due to licensing/privacy, we will provide preprocessing scripts and detailed instructions to reproduce the dataset construction process.
